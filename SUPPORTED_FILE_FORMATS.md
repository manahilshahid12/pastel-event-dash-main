# Supported File Formats

## ✅ Supported Text Formats

The file uploader supports plain text-based formats only:

- **`.txt`** - Plain text files
- **`.md`** - Markdown files
- **`.csv`** - Comma-separated values
- **`.json`** - JSON data files
- **`.js`** - JavaScript files
- **`.html`** - HTML files
- **`.xml`** - XML files
- **`.log`** - Log files
- **`.rst`** - reStructuredText files

## ❌ Not Supported (Binary Formats)

These formats are binary and cannot be read as plain text:

- `.doc` / `.docx` - Microsoft Word
- `.rtfd` - Rich Text Format (macOS)
- `.pdf` - PDF documents
- `.zip` / `.rar` - Compressed archives
- `.jpg` / `.png` / `.gif` - Images
- `.mp3` / `.mp4` - Audio/video

## 🔄 How to Convert Documents

### Convert .doc / .docx to .txt

**Option 1: Using Microsoft Word/Google Docs**
1. Open your .doc or .docx file
2. Select all content (Cmd+A on Mac, Ctrl+A on Windows)
3. Copy (Cmd+C or Ctrl+C)
4. Open a text editor (TextEdit, Notepad, VS Code)
5. Paste the content
6. Save as `.txt` file

**Option 2: Using Google Docs**
1. Upload .doc to Google Drive
2. Open with Google Docs
3. Download as "Plain Text (.txt)"

**Option 3: Using LibreOffice (Free)**
1. Download [LibreOffice](https://www.libreoffice.org/)
2. Open .doc file
3. File → Save As → Format: "Text (.txt)"

### Convert .rtfd to .txt

**On Mac:**
1. Open .rtfd file in TextEdit
2. Select all (Cmd+A)
3. Format → Make Plain Text
4. Save as `.txt`

### Convert .pdf to .txt

**Option 1: Copy text directly**
1. Open PDF in Preview (Mac) or Adobe Reader
2. Select all text
3. Copy
4. Paste into text editor
5. Save as `.txt`

**Option 2: Use online tool**
- [ilovepdf.com](https://www.ilovepdf.com/pdf_to_word)
- [smallpdf.com](https://smallpdf.com/pdf-to-text)

## 💡 Best Practice

For best results, always use **`.txt`** (plain text) or **`.md`** (markdown) files. These ensure:
- ✅ Perfect compatibility
- ✅ No formatting issues
- ✅ Fast upload
- ✅ Easy editing
