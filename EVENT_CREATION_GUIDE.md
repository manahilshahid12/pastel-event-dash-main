## 🎯 How to Create an Event

### **Step 1: Click "New event" Button**
1. Go to **🗓️ Events** tab
2. Click **"New event"** button (top right)

### **Step 2: Fill in Event Details**

**Basic Information:**
- Event Title * (required)
- Description (optional but recommended)
- Venue/Location (where it's happening)
- Event Color (choose a pastel theme)
- Luma URL (if you have one)

### **Step 3: Set Dates & Timeline**

- Start Date * (required)
- Start Time * (required)
- End Date (optional)
- End Time (optional)

### **Step 4: Add Agenda Items**

Plan your event timeline:
1. Type the **time** (e.g., 10:00)
2. Type the **activity** (e.g., "Welcome & Keynote")
3. Click **"+ Add"** button
4. Repeat for each agenda item

**Example:**
```
10:00 - Registration & Coffee
10:30 - Welcome Speech
11:00 - Keynote Presentation
11:45 - Q&A Panel
12:30 - Lunch
```

### **Step 5: Add Speakers & Panelists**

1. Enter **speaker name**
2. Enter **title/company**
3. Enter **bio** (optional)
4. Click **"Add Speaker"** button
5. Add as many speakers as needed

### **Step 6: Add Catering Requirements**

Plan what you need:
1. **Item** - what to order (e.g., "Cupcakes")
2. **Qty** - quantity (e.g., "100")
3. **Notes** - special requests (optional)
4. Click **"Add Item"** button

**Example:**
```
Item: Mini Cupcakes, Qty: 100, Notes: Mix of flavors
Item: Coffee, Qty: 3 thermoses
Item: Vegan Snacks, Qty: 50 pieces
```

### **Step 7: Add Tasks & Responsibilities**

Assign tasks to your team:
1. **Task** - what needs to be done
2. **Assigned to** - who's responsible
3. **Deadline** - when it's due
4. Click **"Add Task"** button

**Example:**
```
Task: Book venue, Assigned: John, Due: 2024-06-01
Task: Order catering, Assigned: Sarah, Due: 2024-06-05
Task: Send invites, Assigned: Mike, Due: 2024-06-03
```

### **Step 8: Submit**

Click **"Create Event"** button to save everything!

## ✏️ How to Edit an Event

### **Method 1: Quick Edit from Calendar**

1. Go to **🗓️ Events**
2. Find your event in the "Upcoming this month" section
3. **Hover over the event card**
4. Click the **✏️ Edit** icon (appears on hover)
5. Update any information
6. Click **"Update Event"**

### **Method 2: Full Edit from Event Details**

1. Click on an event in the calendar
2. You're on the event detail page
3. Each section has **add/edit options**:
   - Requirements checklist
   - Essentials list
   - Guests
   - Agenda
   - Media
   - Speakers
   - Notes

## 📊 What Gets Saved

When you create an event with all the details:

### **Event Table**
- Title, description, location
- Start/end dates and times
- Color theme
- Luma URL
- Created by (user)

### **Speakers Table**
- All speakers you added
- Name, title, bio
- Linked to the event

### **Agenda Table**
- All timeline items
- Time and activity
- Linked to the event

### **Essentials Table**
- Catering items
- Category: "catering"
- Quantity in label
- Linked to the event

### **Requirements Table**
- Tasks/responsibilities
- Format: "Task - Assignee (Due: Date)"
- Status (done/not done)
- Linked to the event

## 💡 Pro Tips

1. **Add agenda first** - Helps determine catering and timing
2. **Invite speakers early** - Add their info in creation form
3. **Set realistic deadlines** - Leave buffer time before event
4. **Use color coding** - Different colors for different event types
5. **Capture everything** - Notes and media right on event page
6. **Team collaboration** - Multiple people can view and add to same event

## 🔄 Workflow Example

### Creating a "Tech Conference"

1. **Create Event**
   - Title: "Annual Tech Summit 2024"
   - Date: June 15, 2024, 9:00 AM - 5:00 PM
   - Location: Convention Center, NYC
   - Color: Sky (calm, professional)

2. **Add Agenda**
   - 09:00 - Registration
   - 09:30 - Keynote: "AI Trends 2024"
   - 10:30 - Break
   - 11:00 - Panel Discussion
   - 12:00 - Lunch
   - etc.

3. **Add Speakers**
   - Jane Smith, CEO Tech Corp, "AI visionary with 20 years..."
   - John Lee, VP Engineering, "Leading innovation at..."

4. **Add Catering**
   - Breakfast pastries, 200 pieces
   - Coffee & tea, 5 thermoses
   - Lunch (box), 150 boxes
   - Snacks, 200 pieces

5. **Add Tasks**
   - Arrange AV setup, Tom, Due: June 1
   - Print materials, Lisa, Due: June 5
   - Send invites, Mike, Due: May 20
   - Confirm speakers, Sarah, Due: June 10

6. **Save** - Everything is now organized in one place!
